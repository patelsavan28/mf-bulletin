# MF Investment Bulletin

Personal Mutual Fund Portfolio Tracker powered by AI.

## Features
- Import holdings via MF Central CAS PDF
- Live NAV auto-fetch from AMFI
- AI-powered portfolio analysis
- Investment news feed
- Data saved to Firebase cloud

## How to Use
1. Open the app
2. Enter your Anthropic API key
3. Download CAS PDF from MF Central
4. Upload PDF — portfolio loads automatically
